### How to install?
1. Clone this repositori
2. Instal dependencies(Makefile)
3. Run games

### Comand for run game
- Check for parity: brain-even
- Calculator: brain-calc
- Greatest common divisor: brain-gcd
- Arithmetic progression: brain-progression
- Prime number: brain-prime
